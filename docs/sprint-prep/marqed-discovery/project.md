# PROJECT | MarQed Discovery Tool

**Priority:** high
**Status:** planned

MarQed Discovery Tool is an AI-powered product discovery assistant that helps product managers decompose project ideas into structured hierarchies of epics, features, and stories. It integrates with Plane for backlog management and uses Claude for intelligent conversation-driven requirements gathering.

Tech stack: React 18 + TypeScript + Vite (frontend), FastAPI + SQLAlchemy + PostgreSQL (backend), Claude API (AI), Plane SDK (project management), assistant-ui (chat components), shadcn/ui (design system).
