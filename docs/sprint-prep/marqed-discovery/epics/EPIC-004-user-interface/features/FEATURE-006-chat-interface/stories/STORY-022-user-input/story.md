# STORY-022 | User Input

**Priority:** high
**Status:** planned
**Depends on:** STORY-020

Implement user input handling with an auto-resizing text area, send button, and keyboard shortcuts. Enter sends the message, Shift+Enter inserts a newline. Empty input is blocked from submission, and the input field clears after sending.

## Acceptance Criteria

- [ ] Text area accepts multi-line input and auto-resizes
- [ ] Enter key sends the message to the backend
- [ ] Shift+Enter inserts a newline without sending
- [ ] Empty or whitespace-only input blocked from submission
- [ ] Input field clears after successful send
