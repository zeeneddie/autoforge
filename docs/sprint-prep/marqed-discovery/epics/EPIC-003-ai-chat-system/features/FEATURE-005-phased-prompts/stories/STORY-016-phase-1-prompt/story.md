# STORY-016 | Phase 1 Prompt

**Priority:** high
**Status:** planned
**Depends on:** STORY-014

Implement the Phase 1 "Context Gathering" prompt template that instructs Claude to collect essential project information: project name, domain, target users, technical preferences, and constraints. Responses are stored as conversation messages linked to the session.

## Acceptance Criteria

- [ ] Phase 1 prompt template sends correctly to Claude API
- [ ] AI asks relevant questions about project name, domain, and users
- [ ] User responses stored as conversation messages in the database
- [ ] Phase 1 completes when sufficient context is gathered
