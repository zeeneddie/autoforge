# STORY-018 | Brownpaper Mode

**Priority:** high
**Status:** planned
**Depends on:** STORY-016

Implement brownpaper mode which loads existing Onboarding output files as context before starting the conversation. The AI presents its understanding of the existing findings for PM confirmation, then interviews the PM about desired changes, additions, or refinements.

## Acceptance Criteria

- [ ] Onboarding output files loaded into Claude conversation context
- [ ] AI references and summarises existing findings accurately
- [ ] PM can confirm, modify, or reject individual findings
- [ ] Modified findings carry forward into subsequent phases
